# Meetings Scheduler

A **modern meeting scheduling platform** designed to simplify calendar coordination and meeting bookings. Key features include:

- **Smart Availability Sharing**: Share available time slots so others can book meetings easily.
- **Google Calendar Integration**: Connect multiple Google accounts and calendars to avoid conflicts.
- **Automatic Meetings Generation**: Instantly create a Google Meet link that can be copied and shared.
- **Interactive Calendar View**: Visual scheduling experience powered by a full calendar interface, using react-big-calendar.
- **Authentication & Billing**: Secure user access with subscription-ready account management.
- **Feedback System**: Dedicated feedback page plus admin panel to review and manage messages
- **User Settings Dashboard**: Control integrations, preferences, and account configuration.
- **Saving Preferences and Other Relevant Data**: Sanity is used behind the hood to store relevant information of the app.

## **Main Tech Stack**

| **Sanity**             | Headless CMS for structured content, feedback storage, and admin workflows            |
